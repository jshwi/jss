A Flask webapp
--------------

Install
-------
.. code-block:: console

    $ docker compose build

Development
-----------
http://localhost:5000

.. code-block:: console

    $ npm run init:dev
    $ docker compose up dev

Production
----------
https://localhost:443

.. code-block:: console

    $ docker compose up production
